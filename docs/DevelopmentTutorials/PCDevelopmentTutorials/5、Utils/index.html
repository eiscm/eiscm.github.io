常用方法

注：tmsCommon 需要在 alias 文件中配置，实际指向的文件是 tms-front/src/common。

1、backToList​

返回到列表，可用于为列表配置开发的详情页

引入​

import { backToList } from "tmsCommon/utils";

参数​

  • backPath :优先级最低的 backPath,默认值。
  • historyState : window.history.state {key:"xxx",state:{backPath:'/xxx/xxx',retainTab:true}}
    • [state.backPath] 跳转路由
    • [state.retainTab] 跳转后是否关闭原本页面，默认关闭，从动态列表打开的 detail 页面会有新开页，从展品标准打开的 detail 页面未必有新开页，因此需要此配置

返回​

  • 无

举例​

列表配置出来的页面，如果需要开发详情页，那么在详情页中可以使用 backToList() 返回到对应的列表页，可以在不知道列表 path 的情况下进行跳转。

backToList();

也可以在普通页面中返回到指定路由页面。

backToList(pathname);

2、getBaiduNamespace​

在页面中异步加载百度地图

引入​

import { getBaiduNamespace } from "tmsCommon/utils";

参数​

  • 无

返回​

  • promise 实例，可以链式调用。

举例​

// 异步加载
// 该方法加载的script有如下几个
// https://api.map.baidu.com/api?v=3.0&ak=${process.env.BAIDU_MAP_AK}&callback=onBMapCallback;
// https://api.map.baidu.com/library/TrafficControl/1.4/src/TrafficControl_min.js
// https://api.map.baidu.com/library/LuShu/1.2/src/LuShu_min.js
// https://api.map.baidu.com/library/CurveLine/1.5/src/CurveLine.min.js
getBaiduNamespace().then(() => {
  // 加载成功后可以拿到 BMap 和 BMapLib 对象，对地图进行初始化、添加控件等操作
  ({ BMap, BMapLib } = window);
  // 根据项目需求进行初始化、添加对应控件，详情请参考百度地图开放平台文档：https://lbsyun.baidu.com/index.php?title=jspopularGL


  // 如果需要动态引入其他文件，可以使用IIFE，如下
  (() => {
    import('./myMarker').then((result) => {
      // do something...
    });
  })();
});
