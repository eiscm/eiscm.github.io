项目启动基本流程​

1.让项目负责人在微信公众平台添加开发者权限

2.下载微信开发者工具

3.拉取项目代码，使用微信开发者工具打开项目。

4.在项目根目录拉取依赖 npm install 或 yarn

5.在工具栏的工具菜单中，点击构建 npm，执行完成后，点击编译，编译小程序。

子应用配置​

1.在 pc 端管理界面的子应用配置菜单新建子应用，并确认登陆角色。子应用配置的路由为htms/wx/subappconfig/query

2.在小程序的配置文件 app.json 中添加子应用页面路由。

  • 产品的子应用，添加到 pages 中

  "pages": [
    "pages/login/login",
    "pages/information/information",
    // ...
    "pages/demo/demo"
  ]

//  ...

  • 客户定制的子应用，添加到客户的子包中

  "subpackages": [
    {
      "root": "SLMPages",// 客户子包的目录
      "pages": [
        // ...
        "pages/demo/demo"
        ]
    },
  ]
